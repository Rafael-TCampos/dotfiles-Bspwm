# Dotfiles BSPWM

Minhas configurações Linux usando **bspwm**, **sxhkd**, **kitty**, **picom**, **polybar** e **LazyVim**.

## Programas utilizados

* bspwm
* sxhkd
* kitty
* picom
* polybar
* feh
* neovim (LazyVim)

## Instalação

Clone o repositório:

```bash
git clone https://github.com/Rafael-TCampos/dotfiles-Bspwm.git
cd dotfiles-Bspwm
```

Execute o script de instalação:

```bash
chmod +x install.sh
./install.sh
```

## Estrutura

```
dotfiles-bspwm
 ├─ bspwm
 ├─ sxhkd
 ├─ kitty
 ├─ picom
 ├─ polybar
 ├─ nvim
 └─ install.sh
```
